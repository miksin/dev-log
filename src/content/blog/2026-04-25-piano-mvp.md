## 概覽

今日完成 `miksin/piano` 的 MVP 開發，由 Hermes Agent 擔任 PM，協調 Engineer 與 QA subagents 進行開發。

## 完成功能

| 功能 | 技術 |
|------|------|
| 2-octave 鋼琴 UI | React + CSS |
| 電腦鍵盤對映 | keydown event listeners |
| Web Audio API 音頻引擎 | ADSR envelope、polyphony |
| MIDI 裝置支援 | Web MIDI API |
| 無障礙屬性 | aria-label、role |
| Unit Tests | Vitest（50 tests） |
| E2E Tests | Playwright（5 tests） |

## 合併的 PR

- **PR #10** — TypeScript 設定 + `.jsx` → `.tsx` 轉換
- **PR #12** — Piano Keyboard UI + Audio Engine
- **PR #13** — Unit Tests 50 個
- **PR #14** — MIDI 支援
- **PR #15** — E2E Tests
- **PR #16** — Vitest config 修正（排除 `e2e/`）
- **PR #17** — Superhuman 深色主題設計

## 技術決策

- 採用 React + Vite + TypeScript strict mode
- 自製 Web Audio API 音頻引擎（不依賴外部函式庫）
- Playwright 與 Vitest 分離設定避免衝突
- 設計風格：Superhuman dark theme（深紫背景、lavender glow on active keys）
